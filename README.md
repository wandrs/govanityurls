# govanityurls

## Create Hugo Project

```bash
hugo new site govanityurls
cd govanityurls
git init
git submodule add https://github.com/appscodelabs/hugo-theme-govanity.git themes/govanity
echo theme = \"govanity\" >> config.toml
```
